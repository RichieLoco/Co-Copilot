## [1.0.0] - 2026-04-16

### Added

- Multi-model chat interface for GitHub Copilot subscriptions
- Dynamic model catalog fetched from GitHub Models API
- Premium request usage tracking via GitHub Billing API
- Projects and persistent conversation history (localStorage)
- File and image upload with drag-and-drop
- Streaming responses with syntax-highlighted code blocks
- Express proxy server to bypass browser CORS
- Docker and docker-compose deployment
- Full documentation for Pi, NAS, Synology, Unraid, TrueNAS, VPS
